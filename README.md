#  Mushroom Classification (Project 3)

## 📌 Overview
This is my **second internship project**, where I developed a Machine Learning model to classify mushrooms as **Edible** or **Poisonous** based on their physical characteristics. The project demonstrates the application of supervised learning techniques for solving real-world classification problems.

## 🚀 Features
- Data preprocessing
- Feature encoding
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- Model evaluation
- Mushroom classification

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn


## 📊 Workflow
1. Load the dataset
2. Preprocess the data
3. Encode categorical features
4. Split data into training and testing sets
5. Train the model
6. Evaluate performance
7. Predict whether a mushroom is edible or poisonous

## 🎯 Objective
To build an accurate machine learning model that classifies mushrooms as edible or poisonous using their physical characteristics.

## 📈 Results
The model achieved high classification accuracy on the test dataset.
